# Query & Transact Advanced API preview sample calls

### getbalances <a name="getBalancesERC20">USDC & DAI (ERC-20)</a>

Retrieve balances for USDC and DAI. Note that this example shows how Query and Transact (QT) Advanced APIs normalize input and accept long, mixed-case address.

**Request:**

```shell
curl https://868605ce-acde-424e-800c-55ab87808268.ethereum.bison.run/ \
  -u username:password \
  -X POST \
  -H 'Content-Type: application/json' \
  -d '{
    "method": "coinbaseCloud_getBalances",
    "params": {
        "addressAndContractList": [
            {
                "address": "0x00000000DF0635793e91d4f8e7426dbd9ed08471186f428D",
                "contract": "0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48"
            },
            {
                "address": "0xdf0635793e91d4f8e7426dbd9ed08471186f428d",
                "contract": "0x6b175474e89094c44da98b954eedeac495271d0f"
            }
        ],
        "blockchain": "Ethereum",
        "network":"Mainnet"
    },
    "id": 1,
    "jsonrpc": "2.0"
}'
```

**Response:**

```json
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": [
    {
      "blockHeight": 15129736,
      "address": "0xdf0635793e91d4f8e7426dbd9ed08471186f428d",
      "nativeAmount": "181205278135673088",
      "nativeUnit": "Wei",
      "tokenBalances": [
        {
          "contract": "0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48",
          "amount": "1780770110",
          "decimals": 6
        },
        {
          "contract": "0x6b175474e89094c44da98b954eedeac495271d0f",
          "amount": "14490426324277149336",
          "decimals": 18
        }
      ]
    }
  ]
}
```
