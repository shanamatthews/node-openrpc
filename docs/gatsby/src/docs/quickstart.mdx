# Quickstart: Get transaction history for an Ethereum address

QT's Advanced Query APIs (beta) allow you to simplify and optimize common request patterns to make developing on the blockchain easier.

Learn how to get the full transaction history for a smart contract or a user address including external, internal, token, ERC-20 and ERC-721 token transfers in a single request.

## Overview

1. Sign in
1. Create project in Coinbase Cloud Console
1. Get API keys
1. Craft your CURL call in the terminal
1. Get results
